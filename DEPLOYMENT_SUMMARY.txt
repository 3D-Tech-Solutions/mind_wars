DEPLOYMENT QUICK REFERENCE
================================================================================

Verify Setup:
  ./scripts/deploy.sh verify

Start Emulator:
  ./scripts/deploy.sh start-emulator

List Devices:
  ./scripts/deploy.sh list-devices

Deploy to Device:
  ./scripts/deploy.sh deploy <device-id> <your-ip>

Build APK (requires clean build):
  flutter clean && rm -rf .dart_tool pubspec.lock
  flutter pub get
  flutter build apk --flavor=alpha \
    --dart-define=FLAVOR=local \
    --dart-define=LOCAL_HOST=192.168.1.100

Install APK:
  adb -s <device-id> install -r build/app/outputs/apk/alpha/release/app-alpha-release.apk

TROUBLESHOOTING QUICK FIXES
================================================================================

Emulator won't boot:
  pkill -f emulator
  sleep 5
  ./scripts/deploy.sh start-emulator

Device not appearing:
  ./scripts/deploy.sh list-devices
  (On physical device: enable USB Debugging in Developer Options)

Compilation errors (undefined method):
  flutter clean
  rm -rf .dart_tool pubspec.lock build
  flutter pub get
  flutter run -d <device-id> ...

Backend connection issues:
  docker ps | grep mindwars
  curl http://localhost:3000/health
